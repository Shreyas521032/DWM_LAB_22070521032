## Prerequisites

- **Python 3.7+**
- **Jupyter Notebook** or **JupyterLab**
- Required Python packages (install via `pip`):
  - `pandas`
  - `numpy`
  - `matplotlib`
  - `seaborn`
  - `scikit-learn`

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```
